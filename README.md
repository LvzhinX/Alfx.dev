## 🚀 Proyecto destacado — Unidas Market

> Marketplace hiperlocal que conecta comercios del barrio con compradores en **Colonias Unidas, Chaco**.

| Característica | Detalle |
|---|---|
| 📱 **Mobile** | React Native (Expo SDK 51) + TypeScript |
| 🗄️ **Backend** | Supabase — PostgreSQL, Auth, Realtime, Storage |
| 🔐 **Seguridad** | Row-Level Security en todas las tablas |
| 🗺️ **Mapas** | OpenStreetMap + Leaflet vía WebView |
| 📷 **QR** | Generación y escaneo por producto/tienda |
| 🔔 **Notificaciones** | In-app con Supabase Realtime |
| ⭐ **Reviews** | Sistema de reseñas con control de privacidad |
| 📊 **Analytics** | Estadísticas de visitas y seguidores por tienda |
